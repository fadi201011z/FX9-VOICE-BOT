# 🎙️ FX9-VOICE Bot v3.0

بوت ديسكورد احترافي بدون Lavalink — يعمل مباشرةً من يوتيوب.

---

## ✨ الميزات

### 🎵 نظام الموسيقى (بدون Lavalink)
| الأمر | الوظيفة |
|-------|---------|
| `/play [query]` | تشغيل أغنية أو قائمة يوتيوب |
| `/skip` | تخطي المقطع الحالي |
| `/stop` | إيقاف الموسيقى وتفريغ القائمة |
| `/pause` | إيقاف مؤقت / استكمال |
| `/volume [1-150]` | ضبط مستوى الصوت |
| `/loop [none/track/queue]` | تغيير وضع التكرار |
| `/queue [page]` | عرض قائمة التشغيل |
| `/nowplaying` | عرض المقطع الحالي |
| `/shuffle` | خلط القائمة عشوائياً |
| `/remove [رقم]` | حذف مقطع من القائمة |

### 🔊 نظام القنوات المؤقتة
| الزر | الوظيفة |
|------|---------|
| 🔒 قفل | منع الانضمام |
| 🔓 فتح | السماح بالانضمام |
| 👥 حد | تحديد عدد الأعضاء |
| ✏️ تسمية | تغيير اسم القناة |
| 👢 طرد | طرد عضو |
| 🙈 إخفاء | إخفاء القناة |
| 👁️ إظهار | إظهار القناة |

**ميزات إضافية:**
- نقل الملكية تلقائياً عند خروج المالك
- حذف القناة تلقائياً عند إخلائها
- لوحة تحكم دائمة في القناة النصية

## 🚀 خطوات التثبيت والتشغيل

### الخطوة 1 — إنشاء بوت Discord

1. اذهب إلى: **https://discord.com/developers/applications**
2. اضغط **New Application** → أدخل اسم `FX9-VOICE`
3. من القائمة اليسرى اختر **Bot**:
   - اضغط **Add Bot** → وافق
   - اضغط **Reset Token** → **انسخ التوكن واحفظه**
   - فعّل هذه الخيارات:
     - ✅ **PRESENCE INTENT**
     - ✅ **SERVER MEMBERS INTENT**
     - ✅ **MESSAGE CONTENT INTENT**
4. من **OAuth2 → URL Generator**:
   - Scopes: ✅ `bot` ✅ `applications.commands`
   - Bot Permissions: ✅ `Administrator`
   - انسخ الرابط وافتحه لدعوة البوت لسيرفرك

---

### الخطوة 2 — تحضير الملفات

```bash
# انتقل إلى مجلد البوت
cd fx9-voice-bot

# انسخ ملف البيئة
cp .env.example .env
```

افتح ملف `.env` وضع بياناتك:

```env
TOKEN=ضع_توكن_البوت_هنا
CLIENT_ID=ضع_application_id_هنا
GUILD_ID=ضع_id_سيرفرك_هنا
```

> **كيف تحصل على CLIENT_ID؟**
> صفحة التطبيق ← General Information ← انسخ **Application ID**
>
> **كيف تحصل على GUILD_ID؟**
> ديسكورد ← Settings ← Advanced ← فعّل Developer Mode
> ثم كليك يمين على السيرفر ← **Copy Server ID**

---

### الخطوة 3 — تثبيت المكتبات

```bash
npm install
```

انتظر حتى ينتهي (قد يستغرق دقيقتين).

> ⚠️ **ملاحظة:** قد تظهر بعض تحذيرات `deprecated` — هذا طبيعي تماماً ولا يؤثر على عمل البوت.
> يجب أن ينتهي التثبيت بـ `added X packages` بدون أي `npm error`.

---

### الخطوة 4 — تسجيل الأوامر (مرة واحدة فقط)

```bash
npm run deploy
```

يجب أن ترى:
```
[Deploy] تسجيل 10 أوامر...
[Deploy] ✅ تم تسجيل 10 أوامر بنجاح
```

### الخطوة 5 — تشغيل البوت

```bash
npm start
```

يجب أن ترى:
```
╔═══════════════════════════════════════╗
║       FX9-VOICE Bot v2.0 Ready!       ║
╚═══════════════════════════════════════╝
```

---

### الخطوة 6 — إعداد القنوات المؤقتة في السيرفر

1. أنشئ **Category** جديدة: `🎙️ القنوات المؤقتة`
2. أنشئ داخلها **Voice Channel**: `➕ انضم لإنشاء قناة`
3. أنشئ **Text Channel**: `🎛️ لوحة-التحكم`
4. نفّذ الأمر في السيرفر:

```
/setup category:[الفئة] join_channel:[القناة الصوتية] text_channel:[القناة النصية]
```

## 🔄 تشغيل البوت دائماً (اختياري)

لتشغيل البوت في الخلفية باستمرار على Linux/VPS:

```bash
# تثبيت PM2
npm install -g pm2

# تشغيل البوت
pm2 start src/index.js --name fx9-voice

# تشغيل تلقائي عند إعادة التشغيل
pm2 startup
pm2 save
```

## ❓ حل المشاكل

| المشكلة | الحل |
|---------|------|
| `Cannot find module` | نفّذ `npm install` مجدداً |
| `Invalid Token` | تأكد من TOKEN في `.env` |
| الأوامر لا تظهر | نفّذ `npm run deploy` وانتظر دقيقة |
| `Missing Permissions` | تأكد من صلاحيات البوت (Administrator) |
| الموسيقى لا تعمل | تأكد أن البوت في قناة صوتية وأن قناة اليوتيوب غير محظورة في منطقتك |
